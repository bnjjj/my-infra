A propos Description My Infra est une application réalisée par l'initiative de Benjamin Coenen et designée par @MrNokte. Elle est distribuée en open source sur Github. Elle n'est en aucune façon affiliée à OVH mais utilise l'API officielle. Cette application utilise exclusivement l'API OVH publique et ne dispose pas d'autre API externe. Je remercie OVH de nous fournir la possibilité de développer avec leur API publique. Si vous avez des idées d'améliorations ou des bugs n'hésitez pas à aller le proposer dans les issues sur le Github de l'application.
